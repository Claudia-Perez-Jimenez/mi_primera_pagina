Este proyecto consiste en crear una página web utilizando HTML y CSS para aprender lo básico aprendido en clase, incluyendo links, imágenes, tablas, listas y un formulario.

Mi página web está dividida en varias secciones; el index, que es el inicio; topic, sobre un tema elegido por mi; mi grado, donde explico detalles del grado elegido y una página solo para la asignatura de fundamentos de la ingeniería informática; about, una página sobre mi; contact, para contactar conmigo y net, para encontrar las páginas de algunos de mis compañeros.

Algunos problemas que me han surgido han sido: poner imágenes, los links, el css...

En conclusión, este proyecto me ha permitido poner en práctica lo aprendido en clase, y desarrollar mis habiidades, además he podido entender como se desarrollan las páginas web en la vida real.
